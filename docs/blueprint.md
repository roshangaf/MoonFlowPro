# **App Name**: ReconFlow Pro

## Core Features:

- Product Lifecycle Tracking: Add, track, and update individual reconditioned items through their entire lifecycle (e.g., received, in repair, tested, listed, sold).
- Detailed Item Records: Store and retrieve comprehensive details for each product, including purchase costs, current condition, repair history, and warranty information.
- Customer Relationship Management: Create and manage customer profiles, link customers to their purchased items, and view their transaction history.
- Automated Reminder Tool: A generative AI tool that analyzes product and customer data to create tailored, automated follow-up reminders for warranty expirations, service dates, or payment due dates.
- Sales and Transaction Tracking: Record sales transactions, associate them with specific reconditioned items and customers, and track sales performance.
- Overview Dashboard: Display a user-friendly dashboard offering a high-level overview of active inventory, upcoming customer reminders, and recent sales activities.

## Style Guidelines:

- Primary color: A deep, professional navy-blue (#262673) conveying trust and sophistication, ideal for headers and primary interactive elements.
- Background color: A subtle, cool light grey-blue (#F5F5F7) for a clean, unobtrusive canvas that ensures content readability.
- Accent color: A vibrant, yet professional electric blue (#52AEFF) for call-to-action buttons, key indicators, and active states to guide user attention.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective, and highly readable design, suitable for all textual content in a business application.
- Utilize a consistent set of crisp, minimalist line icons for intuitive navigation and data representation, enhancing the professional and efficient aesthetic.
- Employ a clean, modular grid-based layout with generous white space, prioritizing clear data organization and easy readability for complex records.
- Implement subtle, fast-acting transitions for state changes and data updates to provide immediate feedback and enhance the fluidity of the user experience.
